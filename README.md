# Fetchium

A reactive query client built on [Signalium](https://signalium.dev) that provides powerful data fetching, caching, and entity management with automatic reactivity.

## Packages

- **[fetchium](./packages/fetchium)** — Core library
- **[docs](./docs)** — Documentation site ([fetchium.dev](https://fetchium.dev))

## Getting Started

```bash
npm install
npm run build
npm test
```
